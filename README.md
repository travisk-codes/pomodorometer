# Pomodorometer

A Pomodoro-focused Time Management Progressive Web App

## MVP features:

- start/stop timer
- toggle between work / break
- user login

  - session based auth

- push notifications
- log of pomodoros
- ability to label pomodoros
- stats

## Stack:

- TypeScript
- React
- Express
- Prisma
- Postgres
- Chakra
- docker
